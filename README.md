# rest-api-express-mongodb-ts
 
 ## Technology
 ```
- Express
- MongoDB
- JWT
- Hash passwords (bcrypt lib)
- Mongoose
- Express Validator
- Role Base
```

## Usage Command
```
yarn (install package)
yarn dev  (for dev)
yarn build (for prod)
```

### Create ENV
```
copy .env.example .env
```
